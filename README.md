# School of the Crossroads to Excellence

A transformative online learning institution dedicated to guiding individuals at pivotal moments in their lives. We provide modern, structured bootcamps in **Leadership**, **Communication**, **Technology**, and **Personal Development**.

![School of Crossroads to Excellence](public/images/logo.png)

## Overview
This platform serves as the digital front door for the School of the Crossroads to Excellence. The website features a highly premium, modern SaaS-style design crafted with deep elegant greens, luxurious golds, and sleek glassmorphism components. It is fully responsive, interactive, and optimized for seamless user experiences.

### Key Features
*   **Immersive User Interface:** Built with clean, modern constraints mimicking high-end educational and corporate interfaces. 
*   **Interactive Regional Parsing:** Program cards dynamically fetch and mutate prices based on a user's selected continent (Africa vs. USA) natively in the UI.
*   **Animated Visuals:** Utilizes `framer-motion` for buttery smooth reveal effects, hover dynamics, and parallax backdrops.
*   **Serverless Contact Routing:** Integrated smoothly with `formsubmit.co` out of the box to instantly pass secure registration applications and contact emails into native inboxes using asynchronous UI state.
*   **Fully Responsive Formats:** Adaptive grids and styling to comfortably scale from ultra-wide desktops directly to tiny mobile configurations.

## Technology Stack
- **Framework:** [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Form Management:** `react-phone-number-input`, Native Fetch API mapping to FormSubmit

## Getting Started

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/school-of-crossroad-to-excellence.git
   ```
2. Navigate into the directory:
   ```bash
   cd school-of-crossroad-to-excellence
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Running Locally
To launch the development server, run:
```bash
npm run dev
```
The application will securely launch on `http://localhost:5173`.
